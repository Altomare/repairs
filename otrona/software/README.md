# Disk format

Specs:
* Double sided
* Double density
* Soft sectored, 10 sectors
* 40 tracks per side
* 48 TPI drive

# Tools

The `diskdefs` file is for use with `cpmtools`. It is from Idkraemer over on vcfed.
[Original link](https://forum.vcfed.org/index.php?threads/otrona-attache.1240458/)
